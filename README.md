# Footprinting tool
----------------------------------------------------
Footpring es una técnica utilizada para recopilar información sobre los sistemas informáticos y las entidades a las que pertenecen.

### Tipo de información que se recolecta
- Nombre de dominio
- Dirección IP
- Registros Whois
- Información DNS
- Sistema operativo utilizado
- Números de teléfono
